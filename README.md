# express-login

## Dependencies
- mysql
  - database name : nodelogin
  - table name : accounts
  - cols of accounts : ( id, username, password, email )
- express
- express-session
- body-parser
- path

## Usage
- terminal : node login.js
- browser : `localhost:3000/regisitor` to create new user.
- browser : `localhost:3000` to login.
